객체 수정 제한하기

적용

  • 객체의 깊은 계층까지 수정을 제한하고 싶을 때

Syntax

메소드 구문 의미
Object.freeze(객체) 객체의 수정을 제한 객체
Object.isFrozen(객체) 객체의 수정을 제한 확인 진릿값

객체는 const도 속성의 추가, 삭제, 변경이 가능하다.

객체 수정을 제한하는 다른 방법

객체의 수정을 제한하는 다른 메소드는 Object.seal(), Object.preventExtension()

  • Object.seal() : 속성의 추가와 삭제 제한, 변경만 가능
  • Object.preventExtenstions(): 속성의 추가 제한. 삭제와 변경만 가능
