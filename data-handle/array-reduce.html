배열 요소 하나로 정리하기

Syntax

메소드 의미 반환
배열.reduce(콜백함수, [초기화*]) 요소(좌->우)를 처리하여 하나의 값 생성 임의 지정값
배열.reduceRight(콜백함수, [초기화*]) 요소(우->좌)를 처리하여 하나의 값 생성 임의 지정값

Syntax -- 콜백함수

구문 의미
((이전요소, 현재요소, 인덱스, 기존배열) => {/*처리내용*/}) 임의의 처리

Examples

예제 1 : 배열에 3개의값을 넣고 합계를 구하는 샘플

예제 2 : 2차 배열을 1차 배열로 반드는 플래트닝Flattening에도 사용할 수 있다.

예제 3 : reduce, reduceRight
