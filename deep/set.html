데이터 중복 없는 Set 사용하기

Syntax

메소드 의미 반환
new Set(반복가능한객체*) Set 객체 초기화 객체(Set)
세트.add(값) 값 등록 객체(Set)
세트.has(값) 값 존재 여부 확인 진릿값
세트.delete(값) 값 삭제 진릿값(삭제 여부)
세트.clear() 값 전체 삭제 객체(iterator)
세트.forEach(콜백) 각 값에 대한 처리 실행 없음

* 생략 가능

속성 의미 타입
세트.size 요소의 수 숫자

Set 객체는 여러 값을 다룬다.
하지만 배열이나 Object와 달리 Index와 키를 사용해 값에 접근할 수 없고 중복되는 값의 설정도 불가능한 것이 특징이다. 세트는 add()나 has()를 사용해 값을 추가하거나 존재 여부를 확인하며, 임의의 값 설정이 가능하다.

values를 사용해 값을 가져오며, 반복자 객체가 반환되므로 for...of 등을 사용해 처리할 수 있다.

중복되는 동일한 값을 설정하면 해당 값은 무시된다.

size 속성으로 세트 내부 요소의 수를 확인
