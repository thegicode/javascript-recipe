reqeustAnimationFrame

Syntax

메소드 의미 반환
reqeustAnimationFrame(함수) 시간 경과에 따른 실행 함수 등록 숫자

reqeustAnimationFrame()은 화면에 각 프레임이 표시되기 저넹 함수를 불러내는 명령으로, 이 메소드를 사용하면 시간 경과에 따른 지속적인 변화를 줄 수 있다. 일반적인 디스플레이는 초당 60프레이미 표시되며, 이는 한 프레임당 16밀리초의 시간이다. 만약 setTimeout()과 setInterval()을 사요해 16밀리초 이하의 시간을 설정한다면, 화면에는 표시되지 않으므로 불필요한 처리가 발생한다. 그러나 reqeustAnimationFrame()을 사용하면 웹에서 불필요한 처리 없이 프레임 표시에 맞춰 애니멩이션 효과를 줄 수 있다.

reqeustAnimationFrame은 주로 WebGl과 HTML 캔버스에서 사용한다.
