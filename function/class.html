클래스 정의하기

Syntax

구문 의미
class 클래스명 {} 클래스를 선언

키워드 class를 사용해 클래스를 선언하며, 클래스명은 Camel Case를 이용하는 것이 일반적

클래스에 constructor()를 삽입하면 초기화 시 해당 메소드가 실행되며, constructor()문의 삽입은 한 번만 가능하다.

클래스 초기화 시 초깃값을 constructor()의 인수로 전달하면 외부 값을 사용할 수 있다.

클래스는 고유의 변수와 함수를 가질 수 있으며, 이 요소를 멤버라고 한다.
