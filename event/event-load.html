DOMContentLoaded를 사용하지 않는 경우
HTML의 head 태그 안에 script 태그가 존재하는 경우 .box 요소의 로딩보다 자바스크립트가 먼저 실행되어 비리므로 .box의 요소를 가져올 수 없다.

load 이벤트는 페이지 내 모든 리소스(사진, 사운도, 동영상 등)의 로딩이 완료된 후 발생한다.
그러므로 DOMContentLoaded보다 시점이 느리다.
페이지가 표시되는 시점에 요소를 조작하고 싶다면 DOMContentLoaded를 사용한다.

defer 속성을 사용하면 HTML 로딩 후 스크립트가 실행된다. DOMContentLoaded보다는 발생 시점이 앞선다.
따라서 defer 속성을 사용하면 DOMContentLoaded 이벤트 설정은 필요가 없다.
