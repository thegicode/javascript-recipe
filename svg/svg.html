SVG 사용하기

SVG : Scalable Vector Grphics

SVG는 2차원 그래픽을 XML로 기술하며, 벡터이므로 해상도에 구애되지 않는 장점을 가졌다. 또한, 다채로운 그래픽을 표현하는 기능도 있다.

viewBox 속성은 SVG의 표시 영역을 정의한다. X 좌표, Y 좌표, 폭, 높이라는 네 가지 값으로 직사각형의 영역을 정의힌다.

SVG의 표시 크기를 width, height 속성으로 설정한다.
viweBox 속성은 SVG 내부 그래픽 좌표를 설정한다. SVG 이미지의 크기를 결정할 때는 width, hegit 속성과 viewBox 속성의 관계도 주의해야 한다.
