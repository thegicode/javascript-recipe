비동기와 Promise 사용하기

Syntax

메소드 의미 반환
new Promise(함수) Promise 생성 Promise 인스턴스
Promise인스턴스.then(함수) 성공 시 콜백 함수 호출 Promise

비동기 처리 작업 기능을 하는 Promise 객체를 사용하면 작업이 용이해질 뿐만 아리나 코드의 가독성도 좋아진다.
fetch()를 시작으로 브라우저의 표준 기능에도 Promise 사용이 증가하고 있으며, await나 async와 같은 기능도 기본적으로 알아두면 좋다.
Promise 생성자의 인수는 비동기 처리 작업을 하는 함수를 지정하며, 함수 내부에서 비동기 처리 완료 메서드인 resolve()를 호출한다.
Promise 인스턴스의 then()응 사용해 resolve() 실행 후 작업을 처리한다.
